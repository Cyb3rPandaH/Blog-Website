In this quick blogpost, I will share the steps that you can follow in order to install Visual Studio Code (1.57.1) *(VSCode)* on a Mac computer using [Brave](https://brave.com/) browser.

## Installing Visual Studio Code
### 1) Downloading VSCode Executable
* Go to [Visual Studio Code](https://code.visualstudio.com/) website.
* Click on `Download Mac Universal`

![](assets/images/blog/installing_vscode_mac/2020-12-03_01_download_vscode_mac.png)

* Click on `Save`.

![](assets/images/blog/installing_vscode_mac/2020-12-03_02_saving_vscode_executable.png)

### 2) Running VSCode Application
* After the file is downloaded, at the `bottom left` of the browser window, open the options and select `Open` to extract the application and open a `finder` windows.

![](assets/images/blog/installing_vscode_mac/2020-12-03_03_open_vscode_application.png)

![](assets/images/blog/installing_vscode_mac/2020-12-03_04_open_vscode_finder_window.png)

* `Right click` on the Visual Studio Code application, and select `open`.

![](assets/images/blog/installing_vscode_mac/2020-12-03_05_open_vscode.png)

* You might get the following message, click on `open`.

![](assets/images/blog/installing_vscode_mac/2020-12-03_06_open_warning.png)

### 3) Visual Studio Code!!

* We are now ready to start working with **VSCode**.

![](assets/images/blog/installing_vscode_mac/2020-12-03_07_vscode_ready.png)

* If you want to change the color theme of the VSCode application, go to the `Code` menu.
* Select `Preferences`  and then select `Color Theme`

![](assets/images/blog/installing_vscode_mac/2020-12-03_08_vscode_color_theme.png)

* Select, for example, `Light (Visual Studio)`.

![](assets/images/blog/installing_vscode_mac/2020-12-03_09_vscode_color_theme_options.png)

* This is how it looks like.

![](assets/images/blog/installing_vscode_mac/2020-12-03_10_vscode_color_theme_light_vscode.png)

### 4) Moving Visual Studio Code to Applications Folder

* Go back to the `finder` window from section 2, and move the Visual Studio Code application to the `applications` folder.

![](assets/images/blog/installing_vscode_mac/2020-12-03_11_vscode_applications_folder.png)

* Do not forget to remove the `zip file` we downloaded at the beginning of this post.

![](assets/images/blog/installing_vscode_mac/2020-12-03_12_vscode_zip_trash.png)
